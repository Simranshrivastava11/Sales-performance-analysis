# Sales Performance Analysis Dashboard

An end-to-end sales analysis project using Python and Power BI to analyze 1,000+ transactions across regions and product categories.

## Tools Used
- Python (Pandas, Matplotlib, Seaborn)
- Power BI
- Excel / CSV

## What This Project Does
- Analyzes sales data across 4 regions and 4 product categories
- Tracks monthly revenue trends throughout 2023
- Calculates profit margins and discount impact
- Visualizes everything in an interactive Power BI dashboard

## Key Findings
- Electronics drove the highest revenue at 35% share
- North region outperformed all other regions in total sales
- Orders with 30%+ discount had significantly reduced profit margins
- Q3 showed peak sales indicating seasonal demand

## Files in This Repository
| File | Description |
|---|---|
| `sales_analysis.py` | Python script for data generation and analysis |
| `sales_data.csv` | Generated sales dataset (1,000 rows) |
| `sales_dashboard.png` | Python matplotlib chart output |
| `sales_dashboard.pdf` | Interactive Power BI dashboard export |
